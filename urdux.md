# UrduX – AI & Human Text Detection Framework

**The Problem:** With the rise of LLMs, detecting AI-generated content in low-resource languages like Urdu became a critical gap in NLP safety research. Existing tools focused on English, leaving Urdu text vulnerable to undetected AI-generated misinformation.

**My Contribution:** I designed and built the entire end-to-end framework — from dataset curation and model selection to deployment architecture.

**Tech Stack:** Python, Transformers (Hugging Face), PyTorch, Custom Urdu Dataset, Fine-tuning Pipeline

**Key Results:**
- Fine-tuned transformer-based classification models on a custom Urdu dataset
- Achieved competitive detection accuracy on AI-generated vs human-written Urdu text
- Contributed academic literature to the domain of low-resource language AI safety
