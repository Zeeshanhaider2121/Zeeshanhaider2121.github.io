# Zeeshan Haider — Portfolio

Static portfolio site built with Jekyll and GitHub Pages.

## Deploy

1. Create a GitHub repository named **`Zeeshanhaider2121.github.io`** (public).
2. Push these files to the `main` branch.
3. Go to **Settings → Pages** and ensure source is set to "Deploy from branch: main".
4. Visit `https://Zeeshanhaider2121.github.io` — your site will be live in 1–2 minutes.

## Local Development

```bash
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```
